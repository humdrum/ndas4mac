Vielen Dank, dass Sie sich für den Kauf einer DataStation maxi z.ul entschieden haben.

Die aktuelle NDAS-Software Version unterstützt keinen gleichzeitigen Schreibzugriff mehrerer Computer. Ein gleichzeitiger Schreibzugriff könnte die Daten auf der DataStation maxi z.ul beschädigen.

Nach der Installation der NDAS-Software können Sie diese über das "NDAS Dienstprogramm"-Symbol im Ordner "Finder/Programme/Dienstprogramme" starten. Dort können Sie mit dem "Registrieren"-Symbol ihre DataStation maxi z.ul anmelden. Die Anmeldedaten der DataStation maxi z.ul werden von der NDAS-Software gespeichert und nicht mehr benötigt.

Die DataStation maxi z.ul ist mit einem FAT32-Dateisystem vorformatiert. Falls Sie diese mit einem NTFS-Dateisystem formatiert haben, müssen Sie die DataStation maxi z.ul im Festplatten-Dienstprogramm ihres Mac OS-X Systems neu partitionieren und formatieren. Nachdem Sie diese neu formatiert haben, wird die DataStation maxi z.ul auf ihrem Desktop angezeigt werden.
