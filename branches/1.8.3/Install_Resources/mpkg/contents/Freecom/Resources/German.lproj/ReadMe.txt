
Vielen Dank für den Kauf des Version OS-X Treibers für Ihr Freecom Netzlaufwerk.

Die aktuelle Version unterstützt nicht den Schreibzugriff für mehrere Benutzer. Wenn versucht wird, von mehreren Computern aus zu auf das Netzlaufwerk zu schreiben, kann dies die auf dem Netzlaufwerk gespeicherten Daten beschädigen.

Beim ersten Anschluss eines NTFS-formatierten (Standard) Netzlaufwerks muss der Benutzer möglicherweise zuerst eine Initialisierung durchführen, um das Laufwerk für ein erweitertes Mac OS Dateisystem oder andere Dateisysteme, auf die Mac OS-X zugreifen kann, zu partionieren und neu zu formatieren. Nach dieser einmalig notwendigen Initialisierung erscheint das Laufwerk auf Ihrem Desktop.

Nachdem Sie im Verzeichnis "/Applications/Utilities" die Anwendung "Freecom NDAS Utility" gestartet haben, geben Sie bitte die (Schreibschlüssel) ein und klicken Sie auf "Add" (Hinzufügen). Den "NDAS ID" und den "Write Key" finden Sie auf der Unterseite des Netzlaufwerks. Nachdem Sie die "Disk ID" und den "Write Key" eingegeben haben, brauchen Sie diese beiden IDs nicht mehr einzugeben.
