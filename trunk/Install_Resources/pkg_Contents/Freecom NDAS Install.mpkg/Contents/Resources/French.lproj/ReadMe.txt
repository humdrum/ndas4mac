Merci d'utiliser le pilote version Freecom Network Drive OS-X.La version actuelle ne permet pas l'accès écriture pour utilisateurs multiples. L'écriture provenant de plusieurs ordinateurs risque d'endommager les données enregistrées sur le lecteur réseau.Les utilisateurs se connectant pour la première fois sur les lecteurs réseau NTFS (standard) doivent éventuellement suivre la procédure d'initialisation pour partitionner et reformater le disque en système de fichiers Mac OS Extended ou un système accessible par Mac OS-X. Après cette procédure unique d'initialisation, le lecteur s'affiche sur le Bureau.Après le démarrage de Freecom NDAS Utility dans le répertoire "/Applications/Utilities", tapez le nécessaire dans "NDAS ID" et "Write Key", informations que vous trouverez en bas du lecteur réseau, puis cliquez sur le bouton "Add". Une fois "Disk ID" et "Write Key" indiqués, il n'est pas nécessaire de les entrer à nouveau.
